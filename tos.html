Privacy Policy for PenguinMod Plus

Last updated: July 22, 2026

PenguinMod Plus (“the Extension”) is a Chrome extension that adds optional addons to the PenguinMod editor. The primary addon, PangLive, enables real-time collaboration on PenguinMod / Scratch projects.

Data we store locally

The Extension uses Chrome’s storage API only to save which addons you have enabled or disabled. This preference data stays in your browser (and Chrome sync, if you use it). We do not operate a server that stores these preferences.

Data handled by PangLive

When you use PangLive collaboration features, the following may be processed between participants over peer-to-peer connections (via WebRTC and the PeerJS library loaded from a CDN):

  • Chat messages you send in a PangLive session
  • Project content and edits shared during collaboration
  • Cursor / presence information shown to other collaborators
  • Network information needed to establish connections (such as IP-related signaling used by WebRTC / PeerJS)

This collaboration data is used solely to provide real-time editing and communication inside PenguinMod. It is not sold.

Third-party services

  • PenguinMod (penguinmod.com / studio.penguinmod.com): the sites where the Extension runs.
  • PeerJS / CDN (for example Cloudflare cdnjs): used to load the PeerJS library and help establish peer connections.

What we do not do

  • We do not sell user data to third parties.
  • We do not use user data for purposes unrelated to the Extension’s single purpose (PenguinMod addons / collaboration).
  • We do not use user data to determine creditworthiness or provide lending.

Contact

For privacy questions about PenguinMod Plus, contact the publisher through the Chrome Web Store listing support / contact details.
